Title: Syntax Highlighting

----

Text:

During my searches through the (link: https://www.11ty.dev/docs/ text: 11ty documentation), I stumbled upon the (link: https://www.11ty.dev/docs/plugins/syntaxhighlight/ text: official syntax highlighting plugin) offered by 11ty. The beauty of this plugin is that it does everything during build-time. This means that there is no need for any client-side JavaScript. ðŸ™Œ

As an example of how this new syntax highlighting looks for now, here you can see a case with the configuration for the syntax highlighting plugin itself:

```js
const syntaxHighlight = require("@11ty/eleventy-plugin-syntaxhighlight");

module.exports = function (eleventyConfig) {
  eleventyConfig.addPlugin(syntaxHighlight);
};
```
